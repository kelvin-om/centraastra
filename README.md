# ⚔️ Centraastra

Centraastra is an interactive web-based engineering tool that allows users to build composite shapes and compute their centroid visually.

---

## 🚀 Features

- 📐 Add multiple shapes:
  - Rectangle
  - Triangle
  - Circular hole

- 🎯 Drag & Drop Interface  
  Move shapes freely with smooth interaction

- 📊 Grid + Coordinate System  
  Includes full positive and negative axes with origin (0,0)

- 🧮 Automatic Centroid Calculation  
  Computes centroid of composite shapes using area method

- 🎨 Visual Representation  
  - Shapes rendered clearly  
  - Hole representation supported  
  - Centroid plotted on canvas  

- 🧹 Shape Management  
  - Select shapes  
  - Delete shapes  

---

## 🧠 Concept Used

Centroid of composite bodies:

x̄ = Σ(Aᵢxᵢ) / ΣAᵢ  
ȳ = Σ(Aᵢyᵢ) / ΣAᵢ  

- Positive area → solid shapes  
- Negative area → holes
